   # Le Baobab Gourmand

Site vitrine d'un restaurant de cuisine africaine moderne basé à Dakar, Sénégal.

Le projet présente un site web statique pour le restaurant **Le Baobab Gourmand**. Il met en valeur la cuisine sénégalaise avec une page d'accueil immersive, un menu, une page de présentation et un formulaire de contact.


## 📌 Sujet choisi

**Sujet A — Site vitrine de restaurant « Le Baobab Gourmand »**

---

## 🛠️ Technologies utilisées

- HTML5 (balises sémantiques)
- CSS3 (styles personnalisés)
- Bootstrap 5 (grille responsive + composants)
- JavaScript Vanilla (interactions, filtres, validation)
- Bootstrap Icons (icônes)

---

## 📄 Pages réalisées

| Page | Fichier | Description |
|------|---------|-------------|
| Accueil | `index.html` | Bannière hero, carousel, plats vedettes, témoignages |
| Menu | `menu.html` | Menu filtrable par catégorie avec photos et prix |
| À propos | `about.html` | Histoire du restaurant, équipe, valeurs |
| Contact | `contact.html` | Formulaire avec validation JS, infos pratiques |

---

## ⚙️ Fonctionnalités JavaScript

- ✅ Filtre dynamique du menu par catégorie (Entrées / Plats / Street Food / Desserts / Jus Naturel/ Boissons Locale)
- ✅ Validation en temps réel du formulaire de contact (champs obligatoires, email valide)
- ✅ Carousel Bootstrap présentant les 3 plats signature
- ✅ Bouton « retour en haut » qui apparaît au scroll
